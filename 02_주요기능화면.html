주요 기능/화면

POKA 시스템은 사용자 친화적인 인터페이스와 직관적인 워크플로우를 제공하여
포토카드 제작 과정을 단순화하고 효율적으로 만들어줍니다.

🔐 로그인/설정

  • 관리자/운영자 인증 - JWT 토큰 기반 보안 인증
  • 환경설정 관리 - 시스템 설정 및 하드웨어 제어
  • 시스템 상태 확인 - 실시간 모니터링
  • 언어 설정 - 한국어/영어/중국어 지원

📱 메인 메뉴

  • 서비스 모드 선택 - ANIMAL/POP 모드 전환
  • 언어 선택 - 다국어 인터페이스
  • 시스템 상태 표시 - 프린터/호퍼 상태
  • 관리자 모드 - 설정 및 관리 기능

📸 사진 처리

  • 이미지 업로드 - QR코드 스캔 기반
  • 편집/미리보기 - 실시간 이미지 편집
  • 스티커/로고 추가 - 다양한 디자인 요소
  • 이미지 최적화 - 자동 리사이징

💳 결제 시스템

  • 현금 결제 - 호퍼 자동 제어
  • LinePay 결제 - 모바일 결제 연동
  • 쿠폰 사용 - 할인 코드 적용
  • 결제 상태 확인 - 실시간 상태 모니터링

사용자 워크플로우

1. 대기 화면 (StandByPage) ↓ 2. 언어 선택 ↓ 3. 메인 메뉴 (MainMenu) ↓ 4. 서비스 모드 선택 (ANIMAL/POP) ↓ 5. QR코드 스캔 (QRCodePage) ↓ 6. 이미지 업로드 대기 ↓ 7. 편집 화면 (EditPage) ↓ 8. 결제 화면 (Payment) ↓ 9. 프린트 화면 (Print) ↓ 10. 완료 화면 (Finish)

화면별 상세 기능

🔄 대기 화면 (StandByPage)

  • 동영상 재생 - 광고 및 안내 영상
  • 언어 선택 버튼 - 한국어/영어/중국어
  • 시스템 상태 표시 - 프린터/호퍼 상태
  • 자동 화면 전환 - 일정 시간 후 메인으로

🎯 메인 메뉴 (MainMenu)

  • ANIMAL 모드 - 동물 테마 포토카드
  • POP 모드 - 팝아트 스타일 포토카드
  • 관리자 모드 - 시스템 설정 및 관리
  • 언어 변경 - 실시간 언어 전환

📱 QR코드 화면 (QRCodePage)

  • QR코드 생성 - 고유 업로드 URL
  • 모바일 안내 - 스마트폰 사용법
  • 업로드 대기 - 이미지 수신 대기
  • 타임아웃 처리 - 자동 화면 전환

✏️ 편집 화면 (EditPage)

  • 이미지 미리보기 - 얼굴/배경 이미지
  • 스티커 선택 - 다양한 스티커 라이브러리
  • 로고 선택 - 그룹별 로고 선택
  • 편집 도구 - 크기 조정, 위치 변경
