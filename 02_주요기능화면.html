📱 주요 화면 상세

🌍

StandByPage (대기 화면)

  • 언어 선택: 한국어, 영어, 중국어
  • 광고 재생: 배경 비디오/이미지
  • 시작 버튼: 터치하여 서비스 시작
  • 자동 대기: 일정 시간 후 자동 시작
🏠

MainMenu (메인 메뉴)

  • 서비스 선택: ANIMAL, POP 카테고리
  • 그룹 선택: 각 카테고리별 그룹
  • 언어 전환: 실시간 언어 변경
  • 뒤로가기: 이전 화면으로 이동
🎨

EditPage (편집 화면)

  • 이미지 업로드: QR코드 스캔
  • 이미지 편집: 크기 조정, 회전
  • 스티커 추가: 다양한 스티커
  • 미리보기: 최종 결과 확인
💳

Payment (결제 화면)

  • 결제 수단: 현금, LinePay, 쿠폰
  • 금액 표시: 선택된 서비스 가격
  • 결제 진행: 실시간 결제 상태
  • 오류 처리: 결제 실패 시 안내
🖨️

Print (프린트 화면)

  • 프린터 상태: 연결 및 용지 확인
  • 출력 진행: 실시간 진행률
  • 대기 화면: 출력 중 사용자 안내
  • 오류 처리: 프린터 오류 시 안내
✅

Finish (완료 화면)

  • 완료 메시지: 서비스 완료 안내
  • 추가 서비스: 추가 제작 옵션
  • 초기화: 시스템 초기 상태로
  • 자동 전환: 대기 화면으로 자동 이동

🎨 사용자 인터페이스 특징

👆

터치 최적화

  • 큰 버튼: 터치하기 쉬운 크기
  • 명확한 피드백: 터치 시 시각적 반응
  • 직관적 아이콘: 이해하기 쉬운 심볼
  • 반응형 디자인: 다양한 화면 크기 대응
🌍

다국어 지원

  • 3개 언어: 한국어, 영어, 중국어
  • 실시간 전환: 즉시 언어 변경
  • 로컬라이제이션: 문화적 특성 반영
  • 일관된 UI: 언어별 동일한 경험
⚡

성능 최적화

  • 빠른 로딩: 최적화된 이미지 및 리소스
  • 부드러운 애니메이션: 60fps 애니메이션
  • 메모리 효율: 효율적인 리소스 관리
  • 오프라인 지원: 기본 기능 오프라인 동작
♿

접근성

  • 높은 대비: 가독성 좋은 색상 조합
  • 큰 폰트: 읽기 쉬운 글자 크기
  • 명확한 네비게이션: 직관적인 화면 이동
  • 오류 안내: 명확한 오류 메시지

📱 반응형 디자인

📱

모바일 최적화

  • 터치 친화적: 터치 타겟 최적화
  • 스와이프 제스처: 직관적인 네비게이션
  • 가상 키보드: 입력 필드 최적화
  • 방향 전환: 세로/가로 모드 지원
💻

데스크톱 지원

  • 마우스 인터랙션: 호버 효과
  • 키보드 네비게이션: Tab 키 지원
  • 고해상도: Retina 디스플레이 지원
  • 윈도우 크기: 반응형 레이아웃
🖥️

태블릿 최적화

  • 중간 크기: 태블릿 전용 레이아웃
  • 멀티터치: 제스처 지원
  • 랜드스케이프: 가로 모드 최적화
  • 스플릿 뷰: 멀티 태스킹 지원
🎯

키오스크 특화

  • 전체 화면: 키오스크 모드
  • 터치 최적화: 큰 버튼과 명확한 UI
  • 자동 시작: 부팅 시 자동 실행
  • 오류 복구: 자동 재시작 기능
