# POKA V2 - 포토카드 제작 서비스

셀픽 사이트와 유사한 모바일 중심의 포토카드 제작 웹 애플리케이션입니다.

## ? 주요 기능

### ? 모바일 최적화
- 반응형 디자인으로 모든 디바이스에서 최적화된 경험
- 터치 친화적인 인터페이스
- PWA(Progressive Web App) 지원으로 앱처럼 사용 가능

### ? 사진 업로드
- 갤러리에서 사진 선택
- 카메라로 실시간 촬영
- 드래그 앤 드롭 지원
- 이미지 압축 및 최적화
- 다중 파일 업로드

### ? 이미지 편집
- 필터 및 효과 적용
- 크롭 및 리사이즈
- 텍스트 및 스티커 추가
- 밝기, 대비, 채도 조정
- 실시간 미리보기

### ?? 갤러리 관리
- 업로드된 이미지 관리
- 즐겨찾기 기능
- 검색 및 필터링
- 그리드/리스트 보기 모드
- 이미지 정보 표시

### ? 사용자 프로필
- 개인 정보 관리
- 사용 통계 확인
- 설정 관리
- 데이터 백업/복원
- 다크 모드 지원

## ?? 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript (ES6+)**: 모듈화된 구조
- **PWA**: Service Worker, Manifest

### 주요 라이브러리
- **Canvas API**: 이미지 처리
- **File API**: 파일 업로드
- **Local Storage**: 데이터 저장
- **IndexedDB**: 대용량 데이터 저장

## ? 프로젝트 구조

```
POKA_V2/

│   └── profile.css         # 프로필 페이지 스타일
├── js/
│   ├── common.js           # 공통 JavaScript
│   ├── index.js            # 메인 페이지 JavaScript
│   ├── upload.js           # 업로드 페이지 JavaScript
│   ├── gallery.js          # 갤러리 페이지 JavaScript
│   └── profile.js          # 프로필 페이지 JavaScript
├── manifest.json           # PWA 매니페스트
├── sw.js                   # 서비스 워커
└── README.md               # 프로젝트 설명
```

## ? 시작하기

### 1. 프로젝트 클론
```bash
git clone [repository-url]
cd POKA_V2
```

### 2. 로컬 서버 실행
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

### 3. 브라우저에서 접속
```
http://localhost:8000
```

## ? PWA 설치

### 모바일 브라우저
1. Chrome/Safari에서 사이트 접속
2. 주소창 옆 설치 버튼 클릭
3. "홈 화면에 추가" 선택

### 데스크톱 브라우저
1. Chrome에서 사이트 접속
2. 주소창 옆 설치 버튼 클릭
3. "POKA V2 설치" 클릭

## ? 주요 기능 상세

### 사진 업로드
- **지원 형식**: JPEG, PNG, GIF
- **최대 크기**: 10MB
- **최소 해상도**: 300x300px
- **자동 압축**: 최대 1024px로 리사이즈

### 이미지 편집
- **필터**: 흑백, 세피아, 빈티지 등
- **조정**: 밝기, 대비, 채도, 색온도
- **효과**: 블러, 샤프, 노이즈
- **텍스트**: 다양한 폰트, 색상, 크기
- **스티커**: 이모지, 아이콘, 프레임

### 갤러리 관리
- **정렬**: 최신순, 오래된순, 이름순, 크기순
- **필터**: 전체, 최근, 즐겨찾기, 편집됨
- **검색**: 파일명, 제목으로 검색
- **보기 모드**: 그리드, 리스트

### 데이터 관리
- **로컬 저장**: 브라우저 로컬 스토리지
- **백업**: JSON 형태로 데이터 내보내기
- **복원**: 백업 파일에서 데이터 가져오기
- **동기화**: 백그라운드 동기화 지원

## ? 성능 최적화

### 이미지 처리
- **Web Workers**: 백그라운드 이미지 처리
- **Canvas 압축**: 클라이언트 사이드 압축
- **지연 로딩**: 이미지 지연 로딩

### 캐싱
- **Service Worker**: 오프라인 지원
- **정적 캐시**: CSS, JS 파일 캐싱
- **동적 캐시**: 이미지 동적 캐싱

### 메모리 관리
- **가비지 컬렉션**: 자동 메모리 정리
- **이미지 해제**: 사용 후 메모리 해제
- **캐시 정리**: 주기적 캐시 정리

## ? 보안

- **권한 관리**: 필요한 권한만 요청

### 개인정보 보호
