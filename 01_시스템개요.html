시스템 개요

POKA-KIOSK는 포토카드 제작을 위한 키오스크 시스템으로, ASP.NET Core 9.0 기반의 웹앱입니다.
다국어 지원, 다양한 결제수단, QR코드 기반 이미지 업로드, 실시간 프린터 제어 등
키오스크 운영에 필요한 모든 기능을 통합 제공합니다.

🖥️ 기술 스택

  • .NET 9.0 - 최신 .NET 프레임워크
  • ASP.NET Core Razor Pages - 웹 애플리케이션 프레임워크
  • SixLabors.ImageSharp - 이미지 처리 라이브러리
  • QRCoder - QR 코드 생성
  • System.IO.Ports - 시리얼 통신 (프린터/호퍼 제어)
  • Newtonsoft.Json - JSON 처리
  • Bootstrap - UI 프레임워크
  • jQuery - JavaScript 라이브러리

👥 주요 대상

  • 현장 사용자 - 키오스크를 직접 사용하는 고객
  • 시스템 관리자 - 시스템 설정 및 관리 담당자
  • 운영자 - 일상적인 운영 및 모니터링 담당자
  • 개발팀 - 시스템 개발 및 유지보수 담당자

🏗️ 아키텍처

  • 클라이언트(키오스크) - 사용자 인터페이스
  • API 서버 - 비즈니스 로직 처리
  • 파일 서버 - 이미지 저장 및 제공
  • 업로드 서버 - 모바일 업로드 처리
