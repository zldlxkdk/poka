POKA 시스템 개요

POKA는 포토카드 제작 키오스크 시스템으로, 사용자가 쉽고 편리하게
개인화된 포토카드를 제작할 수 있는 종합 솔루션입니다.
🎯 포토카드 제작부터 결제까지 원스톱 서비스
📋

시스템 개요

  • 프로젝트 정보: 개발사, 기술 스택, 목적
  • 아키텍처: 시스템 구조 및 구성 요소
  • 기술 스택: 사용된 기술 및 라이브러리
  • 외부 연동: API 서버 및 서비스 연동
🖥️

주요 기능/화면

  • 사용자 인터페이스: 메인 메뉴 및 화면 구성
  • 이미지 편집: 포토카드 편집 기능
  • 결제 시스템: 다양한 결제 수단 지원
  • 프린팅: 고품질 출력 시스템
🔗

URL/API 목록

  • API 엔드포인트: 서버 통신 인터페이스
  • 인증 시스템: 로그인 및 권한 관리
  • 결제 API: 현금/LinePay 결제 처리
  • 파일 업로드: 이미지 업로드 서비스
💾

데이터 구조

  • 데이터베이스: 테이블 구조 및 관계
  • JSON 스키마: 데이터 형식 정의
  • 파일 구조: 이미지 및 설정 파일
  • 캐시 시스템: 성능 최적화
🔐

로그인/보안

  • 인증 시스템: JWT 토큰 기반 인증
  • 권한 관리: 사용자 역할 및 권한
  • 보안 정책: 데이터 보호 및 암호화
  • 세션 관리: 로그인 상태 관리
💳

결제/업로드

  • 결제 수단: 현금, LinePay, 쿠폰
  • QR코드 업로드: 모바일 이미지 업로드
  • 하드웨어 연동: 지폐 인식기, 호퍼
  • 결제 상태: 실시간 결제 모니터링
⚙️

운영/관리

  • 시스템 모니터링: 실시간 상태 확인
  • 설정 관리: 환경 설정 및 구성
  • 로그 분석: 시스템 로그 및 오류
  • 백업/복구: 데이터 백업 및 복원
📚

참고/부록

  • 프로젝트 구조: 파일 및 디렉토리
  • 소스 코드: 주요 소스 파일 목록
  • 설정 파일: 환경 설정 예시
  • 참고 문서: 개발 및 사용 가이드

실제 화면 샘플

  • 인터랙티브 데모: 실제 구동 화면 체험
  • 화면 전환: 다양한 화면 간 이동
