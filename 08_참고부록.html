참고/부록

POKA 시스템의 참고 자료와 부록을 제공합니다.
용어 해설, 참고 문서, 외부 링크, 기타 부가 정보를 포함합니다.

용어 해설

키오스크(Kiosk)

  • 무인 정보 단말기 또는 자동화 기기
  • 사용자 셀프 서비스 제공
  • 포토카드, 티켓, 주문 등 다양한 용도

호퍼(Hopper)

  • 카드 또는 동전 배출 장치
  • 포토카드 완성 후 자동 배출
  • 지폐 인식기와 연동 가능

프린터(Printer)

  • 포토카드 인쇄 장치
  • 이미지 파일을 실물 카드로 출력
  • 상태 모니터링 및 오류 감지

API (Application Programming Interface)

  • 서버와 클라이언트 간 데이터 통신 규약
  • RESTful, JSON 기반 통신
  • 외부 서비스 연동

참고 문서 및 링크

  • ASP.NET Core 공식 문서
  • .NET Core 공식 문서
  • Bootstrap 공식 사이트
  • jQuery 공식 사이트
  • ting.ovh API 서버

코드/설정 예시

appsettings.json 예시: { "Logging": { "LogLevel": { "Default": "Information", "Microsoft.AspNetCore": "Warning" } }, "AllowedHosts": "*", "ConnectionStrings": { "DefaultConnection": "Server=localhost;Database=poka_db;Trusted_Connection=true;" } }
API 호출 예시 (jQuery): $.ajax({ url: "https://pkapi.ting.ovh/api/pay", method: "POST", contentType: "application/json", data: JSON.stringify({ orderId: "ORDER_20241201_001", amount: 5000, paymentType: "cash" }), success: function(res) { if(res.success) alert("결제 성공!"); } });
